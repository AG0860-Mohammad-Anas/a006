# Sample Queries for Jira MCP Host

Here are the 8 required sample queries to test the multi-step reasoning and tool execution of the Groq LLM agent:

1. List all Jira projects
2. Show open issues
3. Show high-priority issues
4. Summarize an issue (e.g., "Summarize A006-2")
5. Show comments (e.g., "Show comments for A006-2")
6. Add a comment (e.g., "Add a comment to A006-2 saying QA validation is pending")
7. Update issue status (e.g., "Update issue A006-2 status to In Progress")
8. Show assigned issues (e.g., "Which issues are assigned to me?")
